# DeviceDetector
A NodeJS and MongoDB application to generate links, that can be sent to target to extracts ts Device Information, IP and Location details

# Steps to install:
  Edit .env configuartion for following environment variable
  DB_URI= ...
  DB_NAME= ...
  Coll_Target= ...
  Coll_User= ...
  SECRET= ...

  npm install
